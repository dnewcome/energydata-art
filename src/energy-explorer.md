# Energy Explorer

Interactive data exploration notebook running Python in the browser via [marimo](https://marimo.io/) + Pyodide (WebAssembly). No server — all computation happens locally in your tab.

Loads on first visit: ~10MB Pyodide runtime + ~30MB CSV data. Cached after that.
